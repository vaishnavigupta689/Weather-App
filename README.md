A **modern Weather App** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.  
The application allows users to search any city and instantly view the **current weather conditions** in a clean and responsive UI.

---

# 📌 Features

✨ Search weather by **city name**  
🌡 Display **temperature in Celsius**  
🌥 Show **weather description and icon**  
💧 Show **humidity level**  
🌬 Display **wind speed**  
❌ Handle **invalid city errors**  
📱 **Responsive UI** with modern glassmorphism design  
⚡ **Fast API fetch using async/await**
